# ERFANR
سیستم جامع کاشف-نسخهِ یِ شبحِ حبشی

## 🚀 ویژگی‌ها

- **داشبورد تعاملی**: نمایش آمار و اطلاعات به صورت زنده
- **تحلیل‌های پیشرفته**: نمودارها و گزارش‌های تحلیلی
- **تنظیمات جامع**: مدیریت کامل حساب کاربری
- **رابط کاربری مدرن**: طراحی زیبا و کاربرپسند
- **حالت تاریک/روشن**: پشتیبانی از تم‌های مختلف
- **انیمیشن‌های روان**: تجربه کاربری بهتر
- **پاسخگو**: سازگار با تمام دستگاه‌ها

## 🛠️ تکنولوژی‌ها

- **React 18** - کتابخانه رابط کاربری
- **TypeScript** - زبان برنامه‌نویسی
- **Tailwind CSS** - فریم‌ورک CSS
- **Framer Motion** - انیمیشن‌ها
- **Lucide React** - آیکون‌ها

## 📦 نصب و راه‌اندازی

### پیش‌نیازها
- Node.js (نسخه 16 یا بالاتر)
- npm یا yarn

### مراحل نصب

1. **کلون کردن پروژه**
```bash
git clone <repository-url>
cd erfanr-app
```

2. **نصب وابستگی‌ها**
```bash
npm install
```

3. **اجرای پروژه**
```bash
npm start
```

4. **ساخت نسخه تولید**
```bash
npm run build
```

## 🔧 تنظیمات توسعه

### ساختار پروژه
```
src/
├── components/          # کامپوننت‌های React
│   ├── Header.tsx
│   ├── Sidebar.tsx
│   ├── Dashboard.tsx
│   ├── Analytics.tsx
│   └── SettingsPanel.tsx
├── App.tsx             # کامپوننت اصلی
├── index.tsx           # نقطه ورود
└── index.css           # استایل‌های اصلی
```

### فایل‌های پیکربندی
- `package.json` - وابستگی‌ها و اسکریپت‌ها
- `tsconfig.json` - تنظیمات TypeScript
- `tailwind.config.js` - تنظیمات Tailwind CSS
- `postcss.config.js` - تنظیمات PostCSS
